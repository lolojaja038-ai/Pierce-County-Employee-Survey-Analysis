# Pierce-County-Employee-Survey-Analysis
Power BI analysis of employee engagement survey responses for Pierce County, WA
Project Overview
This project analyzes 14,725 survey responses from Pierce County government employees to understand engagement levels, department trends, and areas for organizational improvement.
Data Description
Source: Pierce County, WA Open Data
Records: 14,725
Fields: 10 (Department, Role, Survey Questions, Response Scale, etc.)
